# nvim-config

## Key Maps
   
   * Basics
        ** <leader>pv -> Explore
    
    * Telescope
        ** <leader>ff -> Find files
        ** <leader>pg -> Git files
        ** <leader>ps -> Grep find

    * Autocomplete
        ** <C-Space>  -> Toggle autocomplete menu
        ** <C-k>      -> Previous item
        ** <C-j>      -> Next item
